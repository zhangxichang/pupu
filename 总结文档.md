二、 面试题范围总结
综合以上四场面试，可以看出今天的前端面试题主要集中在以下几个技术领域和核心知识点：
1. 核心技术领域
面试题主要围绕前端开发的四大基石展开，具体如下：
技术领域
核心考察知识点
JavaScript (JS) 基础
事件循环、宏任务与微任务、setTimeout、Promise、Proxy、this 绑定、闭包、Set/Map/WeakSet/WeakMap、算法与数据结构（如冒泡排序）。
CSS 基础与布局

盒模型、display 属性（block/Flex/Grid）、position 属性、CSS 全局变量、pointer-events、user-select、calc 函数、伪元素与伪选择器、SVG 创建。
HTML 基础

HTML 文档结构、语义化标签与 div 的使用权衡、元素选择与操作。
前端框架 (Vue)

响应式原理（Object.defineProperty vs Proxy）、核心 API（ref/reactive/computed）、组合式 API 与 setup 函数、effect scope、特殊元素（template/slot/component）、类名绑定。


2. 考察维度与特点
- 基础扎实度 ：面试非常注重候选人对 JavaScript、CSS 和 HTML 核心概念的理解深度和广度，例如事件循环、this 绑定、盒模型等。
- 工程化思维 ：关注候选人对 CSS 变量、主题切换、display 属性在现代应用中的最佳实践等工程化问题的思考。
- 框架深度 ：特别是对 Vue 框架的考察非常深入，不仅涉及基础使用，还包括响应式原理、组合式 API 等进阶内容。
- 实践与项目经验 ：会结合候选人的项目经验，询问其在实际工作中对这些技术点的应用和理解。
- 学习能力与潜力 ：面试过程中会考察候选人的学习态度、理解能力和解决问题的思路，对于不会的问题，更看重其后续的学习意愿和潜力。
3. 重点关注方向
从所有面试题来看，以下几个方向是今天面试的重中之重：
- JavaScript 异步编程模型 ：所有面试都涉及了 setTimeout、Promise、宏任务和微任务，这是前端面试的核心考点。
- CSS 布局与定位 ：特别是 display（Flex/Grid）和 position 属性，是实现复杂页面布局的关键。
- Vue 响应式原理与核心 API ：作为公司的主要技术栈，Vue 的相关知识是决定录用的关键因素之一。
- 实际应用与工程化 ：面试官会结合实际项目场景提问，考察候选人的工程化思维和解决实际问题的能力。
