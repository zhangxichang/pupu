# 噗噗

一个基于现代Web技术构建的跨平台网络聊天应用，既可直接作为Web应用使用，也可打包为原生桌面应用，为用户提供安全、高效、流畅的沟通体验。
![软件宣传图](../assets/promotion_image.png)

## ✨ 核心优势

- **真正的跨平台体验** - 无缝支持 Web、Windows 提供一致且流畅的使用体验
- **即开即用的Web应用** - 无需安装，直接通过浏览器访问 **[噗噗](https://pupu.zhangxichang.com)** 即可开始使用
- **高性能Rust后端** - 应用核心（WebAssembly模块与原生后端）由Rust编写，兼顾卓越性能与内存安全
- **分布式P2P通信** - 采用Kademlia协议构建分布式网络，保障用户互联体验
